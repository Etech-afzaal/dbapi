# DBAPI API Client
This repository provides a client application built with REACT (Next.js) to test and interact with DBAPI

## Key Features:

**Frontend Framework:**
Leverages REACT or Next.js for a modern and performant user interface.

**DBAPI Integration:**
Designed to seamlessly interact with ASP.NET web api.


## Benefits:

**Streamlined Testing:**
Simplifies the process of testing and interacting with your DBAPI.

**Modern Development:**
 Provides a flexible development environment with REACT/Next.js.
